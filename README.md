# vimrc
vimrc and plugins

# First time setup
- Install .vimrc
- curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
- cd ~/.vim/bundle/YouCompleteMe
  python3 install.py
- copy id_rsa id_rsa.pub
